3D Game engine / Model rendering

About:

Creating a game engine written in c++ using OpenGL for graphics rendering. The game engine is so far capable of managing 3d models created in popular modelling software.

Why?

Game engines must be as efficient as possible, both with space and execution speed. This project was started to develop a large scale piece of software that gives experience in working with graphics, mathematics, physics, artificial intelligence and animation from a programming standpoint. This project has helped me massivly improve my skills with making efficient design choices.

The source can be accessed on Github here.
